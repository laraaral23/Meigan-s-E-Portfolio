<p class="divide"> What is CSS? <br> 
~ CSS  stands for Cascading Style  Sheets <br> 
~ CSS describes how  HTML  elements  are  to  be
displayed on screen,  paper,  or  in  other  media.  <br> 
~ CSS saves a  lot  of  work.  It  can  control  the  layout  of
 multiple web pages  all  at  once <br> 
~ External  stylesheets  are stored  in  CSS  files. <br> 
~ CSS is  the  language  we  use to style  a  Web page.  <br> 
~ CSS  separates  content  from  design,  making  websites
easier  to  maintain and more attractive. <br> 
</p> 

<ul class="high"> The Layers of a webpage 
<li> Behavior Layer </li> 
<li> Presentation Layer </li> 
<li> Content Layer </li> 
</ul> 

<h3 class="high" > Content Layer </h3> 
<p class="divide"> define  and  layout  the  text,  images,  animation,  sound, 
 video  and  everything  you  want  to  put  in  a  web page 
 with  the  use  of  tags </p> 
<p class="divide"> is  the  underlying  HTML  code  of  that  page.  Just as a house's  frame  creates  a  strong  foundation  upon  which 
 the  rest  of  the  house  is  built,  a  solid  foundation of HTML 
 creates a platform  upon  which a  website can be created </p>

<h3 class="high"> Presentation Layer </h3> 
<p class="divide"> The presentation layer is where you can define how people see your web page in a browser. </p> 

<h3 class="high"> Behavior Layer </h3> 
<p class="divide"> The behavior layer is where you have a real-tirm user interaction with the web page. The intyeractiob may be simple validation or input or filling out a form or even as big as web-based programs. </p>

<ul class="high"> Benefits of using CSS 
<li> CSS saves a lot of work </li> 
<li> Easy maintenance </li> 
<li> Adding style to webpage makes it more pleasing to the eye. </li>
</ul>
